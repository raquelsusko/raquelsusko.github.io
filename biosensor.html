Electrochemical Biosensor

Real-time detection of cardiovascular biomarkers

Home Robotic Hand Biosensor

Overview

This biosensor project used an STM32WB55 microcontroller and an AD5940 potentiostat to measure electrochemical reactions of BNP and NT-proBNP biomarkers. I developed the firmware, configured SPI communications, and implemented data logging.

Technologies Used

  • STM32WB55 + STM32CubeIDE
  • AD5940 potentiostat
  • SPI communication
  • Electrochemical signal processing
