Home Robotic Hand Biosensor

Overview

This project involved the development of a multi-finger robotic hand using EMG sensor inputs and microcontroller-based control systems. I focused on mechanical design, signal acquisition, and embedded programming to emulate realistic hand movement.

Technologies Used

  • Arduino microcontroller
  • EMG signal acquisition
  • Servo control
  • 3D-printed hand structure
