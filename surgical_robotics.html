System Overview: Haptic Teleoperation

I developed a ROS 2-based teleoperation system that enables intuitive control of a UR5 robotic arm using Force Dimension haptic devices. The system was specifically designed to facilitate high-precision data collection for imitation learning applications in partial nephrectomy procedures.

  • PID Control: Engineered trajectory smoothing to ensure surgical-grade precision.
  • Multi-Modal Integration: Built a custom 3-pedal foot interface for clutch, vacuum, and recording control.
  • Data Synchronization: Captured 6-DOF poses, velocities, and RGB-D point cloud data for ML training.
