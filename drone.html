Overview

This project focused on developing an end-to-end system for an aerial robot to autonomously identify, approach, and retrieve objects. This required bridging the gap between high-level computer vision and low-level flight control.

  • Integrated an SO101 robotic arm onto a drone chassis, managing weight distribution and center-of-gravity shifts.
  • Implemented the SmolVLA architecture to enable visual-language
